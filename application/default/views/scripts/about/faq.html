よくある質問

Q : 会員登録したい

A : ログインページの「サインアップ」ボタンを押してください。次のページに出てきた、必要事項を入力して「完了」ボタンを押します。これで会員登録は完了です。

Q : アイテムを投稿したい

A : まずは、会員登録をしてください。そのあと、アプリのTOPページに「アイテム投稿」ボタンがございますので、こちらより投稿をして下さい。既存の写真データでも写真を撮っての投稿も可能です。また、フレーム・フィルターもございますので是非ご利用ください。

Q : 投稿したアイテムを削除したい

A : マイページ内にご自分の投稿したアイテムの一覧ページを確認する「投稿アイテム一覧」ページがございます。こちらの中から、削除したいアイテムを選択していただきます。右上に「削除」ボタンがございますのでそこから削除を行ってください。

Q : 不正なアイテムを削除して欲しい

A : アイテム詳細の左下に三角のアイコン「通報」ボタンがございます。こちらを押すことで管理者に通告できます。即急を要すものは下記のアドレスにご連絡をください。info@cawaii.jp

Q : 投稿されているアイテムを購入したい

A : アイテム詳細を見ていただき、「ショップ」から直接サイトで購入していただけます。URLを設定していないものは閲覧不可能ですのでご注意ください。

Q : アイテムのcawaiiね!を削除したい

A : 現在アプリにそういった機能は備わっておりません。申し訳ございません。下記のアドレスにご連絡をください。info@cawaii.jp

Q : 掲示板に投稿した内容を削除したい

A : 現在アプリにそういった機能は備わっておりません。申し訳ございません。下記のアドレスにご連絡をください。info@cawaii.jp

Q : 登録情報を変更したい

A : マイページ内にございます、「プロフィールを確認・変更する」から、ニックネーム・写真をアップロード・生年月日を公開、非公開の選択・コメントの編集をおこなうことができます。

Q : SNSと連携したい

A : cawaiiでは、Facebook・twitter・mixiのSNSと連携をしているので、いずれにも同時に写真を投稿することができます。SNSを連携するためには、マイページの「SNS連携」という部分をタップしていただきます。そのあとは、必要な情報を書き込んでいただき連携完了です。

Q : 退会したい

A : iphoneアプリを削除していただく、または、下記のアドレスにご連絡をください。info@cawaii.jp
